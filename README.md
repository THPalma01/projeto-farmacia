# 💊 Sistema de Farmácia (Microsoft Access)

## 🧠 Descrição

Sistema desenvolvido em Access para controle de produtos e gestão básica de uma farmácia.

## 🎯 Objetivo

Aplicar conceitos de banco de dados em um cenário real.

## 🛠️ Tecnologias

* Microsoft Access

## ⚙️ Funcionalidades

* Cadastro de produtos
* Controle de estoque
* Organização de dados

## 🧠 Conceitos aplicados

* Modelagem de dados
* Consultas
* Estruturação de tabelas

## ▶️ Como executar

Abrir o projeto no Microsoft Access.
